## ⚙️ Características principales

✅ Compatible con **WhatsApp Multi-Device (MD)**  
✅ Sistema modular de **plugins y comandos personalizados**  
✅ Soporte para **descargas (YouTube, TikTok, Instagram)**  
✅ Comandos de **entretenimiento y RPG**  
✅ Integración con **base de datos PostgreSQL**  
✅ Sistema de **auto-reconexión y guardado de sesión**  
✅ Protección anti-spam y anti-crash  
✅ Soporte para **Termux**, **VPS**, **Replit** y **Docker**

---

## 🧠 Requisitos previos

Antes de iniciar, asegúrate de tener instalado:

- [Node.js](https://nodejs.org/) versión **18 o superior**
- [Git](https://git-scm.com/)
- [PostgreSQL](https://www.postgresql.org/)
- Una cuenta activa de WhatsApp

---

## 🚀 Instalación

```bash
# 1. Clonar el repositorio
git clone https://github.com/dvnmendoza123-gif/AGUILA-BOT-MD.git

# 2. Entrar al directorio
cd AGUILA-BOT-MD

# 3. Instalar dependencias
npm install

# 4. Configurar variables de entorno (.env)
# Puedes copiar el archivo .env.example y poner tus datos

# 5. Iniciar el bot
node index.js
